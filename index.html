Featured Project: RPi WireGuard VPN Gateway

This project connects my laptop to a Raspberry Pi 4 ARM (running Kali Linux) via a WireGuard VPN, using a VPS as a central server. The Raspberry Pi manages multiple WiFi interfaces, one configured as an AP routing traffic through the VPN, all controlled remotely via VNC.

  • Advanced routing across multiple interfaces (wlan0, wlan1, wlan2, wg0)
  • Remote graphical access via VNC
  • Secure access from any location
