🔐 Proyecto destacado: VPN con WireGuard y RPi

Este proyecto conecta mi laptop con una Raspberry Pi 4 ARM corriendo Kali Linux mediante una VPN con WireGuard, usando un VPS como servidor central. La Raspberry tiene varias interfaces WiFi, una configurada como AP con salida por la VPN, y se controla remotamente por VNC.

  • ✅ Enrutamiento avanzado con múltiples interfaces (wlan0, wlan1, wlan2, wg0)
  • ✅ Acceso gráfico remoto vía VNC
  • ✅ Acceso seguro desde cualquier lugar
